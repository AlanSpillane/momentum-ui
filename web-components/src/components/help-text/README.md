# Help Text

### Attributes

- `message` : String - message text
- `messageType` : String - set type to style for `success`, `warning`, `error`

### Slots

#### Unnamed Slots

The Unnamed `<slot>` will render any content you pass between the `<md-help-text></md-help-text>` tags.

```html
<md-help-text>
  <p>Content</p>
</md-help-text>
```
